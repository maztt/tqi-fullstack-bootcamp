GIT/GITHUB: permite ver os versionamentos, enxergar uma linha do tempo do código.

-> dir = lista todos os diretórios
-> cd (change directory) = navega nas pastas
  -> cd Windows = acessa pasta Windows
  -> cd .. = volta uma pasta
-> cls = limpa o terminal
-> "TAB" = complementa o nome do arquivo da pasta
-> mkdir (make directory) [nome] = cria a pasta com o nome escolhido
  -> silence on success
-> echo hello > hello.txt = cria-se o arquivo na pasta
-> del = deleta arquivos do diretório, ex: " 'del workspace' deletaria os arquivos da pasta workspace "
-> rmdir (remove directory) [nome] /S /Q = deleta o diretório por completo
